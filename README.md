### Installation Steps

1. Clone the repository

```Bash
git clone https://github.com/dipsam/PotterHead_Quiz.git
```

2. Change the working directory

```Bash
cd PotterHead_Quiz/src
```

3. Run the app using your terminal

```Bash
node index.js
```
